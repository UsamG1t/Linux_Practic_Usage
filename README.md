Данный репозиторий является хранилищем материалов, используемых в рамках курса «Практическое использование Linux» программы дополнительного образования «Специалист по сетевым технологиям»

В рамках курса изучаются свойства дистрибутива Linux, которые необходимы для повседневного использования ОС в роли технологического компонента многомашинных систем — при моделировании сетевых структур, параллельных вычислений, исследования безопасности и т. п.

## Темы, рассматриваемые в курсе

1. [Загрузка ОС, EFI](./Boot-EFI/)
2. Работа в командной строке
3. Структура файловой системы
4. Принципы настройки сетевых служб
5. Управление процессами
