Данный репозиторий является хранилищем материалов, используемых в рамках курса «Практическое использование Linux» программы дополнительного образования «Специалист по сетевым технологиям»

В рамках курса изучаются свойства дистрибутива Linux, которые необходимы для повседневного использования ОС в роли технологического компонента многомашинных систем — при моделировании сетевых структур, параллельных вычислений, исследования безопасности и т. п.

## Темы, рассматриваемые в курсе

1. [Загрузка ОС, EFI](./Boot-EFI/)
2. [Работа в командной строке](./Terminal-Shell)
3. [Управление процессами](./Process-Management)
4. [Структура файловой системы](./FileSystem-Mounting)
5. [Принципы настройки сетевых служб](./SystemServices)

## Свободное чтение

Ссылка на версию репозитория [для чтения](https://usamg1t.github.io/Linux_Practic_Usage).

**Обратите внимание:**
1. Репозиторий находится _на стадии доработки_. На данный момент для чтения доступны материалы по темам **№2, №3, №4, №5**. Тема №1 находится в формате лекторского представления, но не свободного чтения.
2. Для ознакомления с материалами переходите на страницы соответствующих тем и открывайте «Текстовую версию» лекции. После ознакомления с материалом рекомендуется проделать самостоятельную работу, представленную на странице темы.
